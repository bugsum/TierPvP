# ⚔️ What is TierPvP?

TierPvP is an intense arena-based PvP minigame where every elimination upgrades your loadout. Players begin with a basic weapon, such as a wooden axe and progress through increasingly powerful kits as they secure kills.

### The objective is simple:
- Eliminate opponents
- Advance through weapon tiers
- Reach the final tier
- Claim victory before everyone else

Matches are designed to be quick, chaotic, and highly replayable.

Whether your arena contains narrow bridges, dangerous voids, water traps, or vertical combat zones, every fight will be a test of pvp skill, movement, and positioning.

## 🎮 Core Gameplay
- Players start at Tier 1 with a starter weapon
- Every kill advances the player to the next tier
- Each tier can have completely unique kits and equipment
- Falling into the void, or water results in death
- First player to complete all tiers wins the match

## ✨ Features

1. **Fast Paced PvP**: Designed for nonstop action with rapid respawns and continuous combat.
2. **Fully Customizable Tier System**: Customize every progression level using the tier configuration.
3. **Arena System**: Create and manage multiple arenas directly in-game.
4. **Arena Shuffling**: Smart Arena Shuffling algorithm so the same arena never comes twice in a row.
5. **Highly Configurable**: Nearly every aspect of the plugin can be adjusted to fit your server. Including the gameplay, config OR tiers.
6. **Performance Focused**: Built to provide smooth gameplay during fights and high player activity.

# 🏆 Why TierPvP?

Unlike traditional PvP minigames, TierPvP rewards consistency and momentum. Every kill changes your gear, forcing players to adapt their combat style throughout the match.

No two fights feel the same.

# 🔨 Commands & Permissions

| Commands | Description | Permissions |
| -------- | -------- | -------- |
| /tierpvp stats | Check the stats for a player | tierpvp.stats |
| /tierpvp forcestart | Forcefully restart a new game | tierpvp.admin |
| /tierpvp reload | Reload the plugins config files | tierpvp.admin |
| /tierpvp create | Create an arena | tierpvp.admin |
| /tierpvp delete | Delete an arena | tierpvp.admin |
| /tierpvp setspawn | Set the spawn point for an arena | tierpvp.admin |
| /tierpvp setzone | Set two coordinates for the arena's spawn. | tierpvp.admin |
| /tierpvp confirmzone | Confirm Arena's spawn zone selection | tierpvp.admin |
| /tierpvp cancelzone | Cancel Arena's spawn zone selection | tierpvp.admin |
| /tierpvp setvoid | Set the void Y level of an arena | tierpvp.admin |
| /tierpvp list | List all arenas | tierpvp.admin |
| /tierpvp info | Get info for a specific arena | tierpvp.admin |
